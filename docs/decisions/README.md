# Architecture Decision Records

This package is distilled from a larger private codebase, and ADR numbering
continues from that project's sequence rather than restarting at 001. Only the
decisions relevant to this public package are mirrored here — which is why the
directory currently starts at [ADR-006](006-llm-protocol-abstraction.md). The
numbers are kept stable (not renumbered) so existing references to an ADR keep
resolving.
